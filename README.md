# Learning Path

Animated, from-zero learning roadmaps for three modern engineering tracks:

- **AI Engineer** — math foundations → deep learning → LLMs in production
- **Agentic Engineer** — prompt design → tool use → multi-agent systems
- **Full-Stack Developer** — HTML/CSS/JS → React + Node → cloud deploy

Every track ships with a **4-week program** and a curated **interview-prep**
question bank.

## Stack
- React 18 + Vite
- Framer Motion for animation
- CSS variables for light/dark theming (toggle top-right)

## Develop
```bash
npm install
npm run dev
```

Then open the URL Vite prints (default http://localhost:5180).

## Build
```bash
npm run build
npm run preview
```
